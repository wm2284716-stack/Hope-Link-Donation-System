Hope-Link Donation System
🌟 Overview
Hope-Link is a desktop application developed using C++ and the Qt Framework. The platform serves as a bridge between donors and charitable causes, allowing for efficient management of donations and donor information.

🚀 Features
Donor Registration: Easily register new donors with their essential information.

Donation Management: Track different types of donations (Financial, Supplies, etc.).

User-Friendly Interface: Built with Qt GUI for a smooth and interactive user experience.

Data Organization: Uses Object-Oriented Programming (OOP) principles to ensure clean and maintainable code.

🛠️ Built With
Language: C++

Framework: Qt Framework (GUI)

IDE: Qt Creator

📂 Project Structure
main.cpp: Entry point of the application.

mainwindow.h/.cpp: Handles the main UI logic and interactions.

donation.h: Defines the core data structures and logic for the donation system.

hopelink.pro: Configuration file for the Qt project.

🔧 How to Run
Ensure you have Qt Creator and a C++ compiler (like MinGW or MSVC) installed.

Clone this repository or download the source code.

Open the hopelink.pro file in Qt Creator.

Build and run the project.
